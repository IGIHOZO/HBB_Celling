EXML to XML Converter REST API

← Back to Converter

Overview

The EXML to XML Converter provides a REST API for converting Erlang XML format (EXML) to standard XML. The API accepts EXML strings and returns properly formatted XML.

EXML Format

EXML Structure:

{element, "tagname", [{attribute, "name", "value"}], [children...]}
  • Elements: {element, "tagname", attributes, children}
  • Attributes: {attribute, "name", "value"}
  • Text content: "text string"
  • Children: List of elements or text strings

API Endpoints

POST /convert/

Convert EXML to XML

Request Body:
{ "exml": "{element, \"root\", [], []}" }
Response (Success):
{ "success": true, "xml": "\\n", "exml": "{element, \"root\", [], []}" }
Response (Error):
{ "success": false, "error": "Conversion failed: Invalid EXML format" }

GET /examples/

Get example EXML inputs

Response:
{ "success": true, "examples": [ { "name": "Simple Element", "exml": "{element, \"root\", [], []}" }, { "name": "Element with Attributes", "exml": "{element, \"person\", [{attribute, \"id\", \"123\"}], []}" } ] }

Usage Examples

JavaScript/Fetch

const response = await fetch('/convert/', { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-CSRFToken': getCookie('csrftoken') }, body: JSON.stringify({ exml: '{element, "person", [{attribute, "name", "John"}], []}' }) }); const data = await response.json(); console.log(data.xml);

Python/Requests

import requests response = requests.post('http://localhost:8000/convert/', json={ 'exml': '{element, "person", [{attribute, "name", "John"}], []}' } ) data = response.json() print(data['xml'])

curl

curl -X POST http://localhost:8000/convert/ \ -H "Content-Type: application/json" \ -d '{"exml": "{element, \"person\", [{attribute, \"name\", \"John\"}], []}"}'

Error Codes

Error Description Example
Empty input EXML input cannot be empty "EXML input cannot be empty"
Invalid JSON Request body is not valid JSON "Invalid JSON data"
Invalid EXML EXML format is malformed "Conversion failed: Unsupported EXML format"
Parse error Error parsing EXML structure "Conversion failed: Invalid tag format"

Response Format

Success Response

{ "success": true, "xml": "formatted_xml_output", "exml": "original_exml_input" }

Error Response

{ "success": false, "error": "error_message" }
📝 Note

The API expects EXML strings in Erlang tuple format. Make sure to properly escape quotes when sending JSON requests. The XML output is formatted with proper indentation for readability.
